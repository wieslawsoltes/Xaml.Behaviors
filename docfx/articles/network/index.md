# Network Interactions

The `Avalonia.Xaml.Interactions.Network` namespace provides behaviors for simple network scenarios without writing code-behind.

## Actions

- [HttpRequestAction](http-request-action.md): Sends an HTTP request and exposes the response content and status code as bindable properties.

## Triggers

- [NetworkInformationTrigger](network-information-trigger.md): Listens to connectivity changes and runs actions when network availability changes.
