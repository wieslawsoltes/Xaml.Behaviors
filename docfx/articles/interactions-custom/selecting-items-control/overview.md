# SelectingItemsControl

The `Avalonia.Xaml.Interactions.Custom` package provides behaviors for working with `SelectingItemsControl` in Avalonia.

## Behaviors

| Behavior | Description |
| --- | --- |
| [SelectingItemsControlEventsBehavior](selecting-items-control-events-behavior.md) | An abstract base class for behaviors that listen to the `SelectingItemsControl.SelectionChanged` event. |
| [SelectingItemsControlSearchBehavior](selecting-items-control-search-behavior.md) | Filters `SelectingItemsControl` items based on the text of a search box. |
