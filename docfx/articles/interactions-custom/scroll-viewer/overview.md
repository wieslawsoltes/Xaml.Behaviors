# ScrollViewer

The `Avalonia.Xaml.Interactions.Custom` package provides actions, behaviors and triggers for working with `ScrollViewer` in Avalonia.

## Actions

| Action | Description |
| --- | --- |
| [ScrollToOffsetAction](scroll-to-offset-action.md) | Scrolls a `ScrollViewer` to the specified offsets when executed. |

## Behaviors

| Behavior | Description |
| --- | --- |
| [HorizontalScrollViewerBehavior](horizontal-scroll-viewer-behavior.md) | Enables horizontal scrolling of a `ScrollViewer` using the mouse wheel. |
| [ScrollViewerOffsetBehavior](scroll-viewer-offset-behavior.md) | Sets the `ScrollViewer.Offset` of the associated `ScrollViewer`. |
| [ViewportBehavior](viewport-behavior.md) | Listens for the associated element entering or exiting the parent `ScrollViewer` viewport. |

## Triggers

| Trigger | Description |
| --- | --- |
| [ScrollChangedTrigger](scroll-changed-trigger.md) | Executes actions when the `ScrollViewer.ScrollChanged` event occurs. |
