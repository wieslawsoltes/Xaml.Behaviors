# TabControl

The `Avalonia.Xaml.Interactions.Custom` package provides behaviors, actions and triggers for working with `TabControl` in Avalonia.

## Actions

| Action | Description |
| --- | --- |
| [TabControlNextAction](tab-control-next-action.md) | Advances the target `TabControl` to the next tab. |
| [TabControlPreviousAction](tab-control-previous-action.md) | Moves the target `TabControl` to the previous tab. |

## Behaviors

| Behavior | Description |
| --- | --- |
| [TabControlKeyNavigationBehavior](tab-control-key-navigation-behavior.md) | Enables keyboard navigation for a `TabControl` using arrow keys. |

## Triggers

| Trigger | Description |
| --- | --- |
| [TabControlSelectionChangedTrigger](tab-control-selection-changed-trigger.md) | Executes actions when the associated `TabControl` changes selection. |
