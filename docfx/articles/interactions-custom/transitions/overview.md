# Transitions

The `Avalonia.Xaml.Interactions.Custom` package provides behaviors, actions and triggers for working with `Transitions` in Avalonia.

## Actions

| Action | Description |
| --- | --- |
| [AddTransitionAction](add-transition-action.md) | Adds a `TransitionBase` to the `Avalonia.Animation.Transitions` collection on the target element. |
| [ClearTransitionsAction](clear-transitions-action.md) | Clears the `Avalonia.Animation.Transitions` collection. |
| [RemoveTransitionAction](remove-transition-action.md) | Removes a `TransitionBase` from the `Avalonia.Animation.Transitions` collection on the target element. |

## Behaviors

| Behavior | Description |
| --- | --- |
| [TransitionsBehavior](transitions-behavior.md) | Sets the `Avalonia.Animation.Transitions` collection on the associated control when attached. |

## Triggers

| Trigger | Description |
| --- | --- |
| [TransitionsChangedTrigger](transitions-changed-trigger.md) | Executes actions whenever the `Avalonia.Animation.Transitions` collection changes. |
