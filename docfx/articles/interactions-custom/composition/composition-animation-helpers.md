# CompositionAnimationHelpers

`CompositionAnimationHelpers` is an internal static class that provides utility methods for working with Avalonia's Composition API. It handles tasks such as:

-   Creating and starting vector and scalar keyframe animations.
-   Managing center points and anchor points for visuals.
-   Converting degrees to radians.

It is used by the various animation helper classes to execute the animations.

> **Note:** This class is internal and not intended for direct use by consumers of the library.
