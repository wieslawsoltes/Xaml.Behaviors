# ExecuteCommandOnPullGestureBehavior

A behavior that executes a command when a pull gesture occurs.

## Usage

```xml
<ListBox xmlns="https://github.com/avaloniaui"
         xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml">
    <Interaction.Behaviors>
        <ExecuteCommandOnPullGestureBehavior Command="{Binding PullGestureCommand}" />
    </Interaction.Behaviors>
</ListBox>
```
