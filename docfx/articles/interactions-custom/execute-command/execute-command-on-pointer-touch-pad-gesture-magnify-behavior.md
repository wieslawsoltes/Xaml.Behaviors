# ExecuteCommandOnPointerTouchPadGestureMagnifyBehavior

A behavior that executes a command when a touch pad magnify gesture occurs.

## Usage

```xml
<Image xmlns="https://github.com/avaloniaui"
       xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml">
    <Interaction.Behaviors>
        <ExecuteCommandOnPointerTouchPadGestureMagnifyBehavior Command="{Binding MagnifyCommand}" />
    </Interaction.Behaviors>
</Image>
```
