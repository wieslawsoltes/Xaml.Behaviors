# ExecuteCommandOnPointerTouchPadGestureSwipeBehavior

A behavior that executes a command when a touch pad swipe gesture occurs.

## Usage

```xml
<Image xmlns="https://github.com/avaloniaui"
       xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml">
    <Interaction.Behaviors>
        <ExecuteCommandOnPointerTouchPadGestureSwipeBehavior Command="{Binding SwipeCommand}" />
    </Interaction.Behaviors>
</Image>
```
