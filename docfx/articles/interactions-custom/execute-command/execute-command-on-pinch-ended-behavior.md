# ExecuteCommandOnPinchEndedBehavior

A behavior that executes a command when a pinch gesture ends.

## Usage

```xml
<Image xmlns="https://github.com/avaloniaui"
       xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml">
    <Interaction.Behaviors>
        <ExecuteCommandOnPinchEndedBehavior Command="{Binding PinchEndedCommand}" />
    </Interaction.Behaviors>
</Image>
```
