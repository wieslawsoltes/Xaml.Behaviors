# ExecuteCommandOnScrollGestureInertiaStartingBehavior

A behavior that executes a command when a scroll gesture inertia starts.

## Usage

```xml
<ScrollViewer xmlns="https://github.com/avaloniaui"
              xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml">
    <Interaction.Behaviors>
        <ExecuteCommandOnScrollGestureInertiaStartingBehavior Command="{Binding ScrollGestureInertiaStartingCommand}" />
    </Interaction.Behaviors>
</ScrollViewer>
```
