# ExecuteCommandOnTextInputBehavior

A behavior that executes a command when text input occurs.

## Usage

```xml
<TextBox xmlns="https://github.com/avaloniaui"
         xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml">
    <Interaction.Behaviors>
        <ExecuteCommandOnTextInputBehavior Command="{Binding TextInputCommand}" />
    </Interaction.Behaviors>
</TextBox>
```
