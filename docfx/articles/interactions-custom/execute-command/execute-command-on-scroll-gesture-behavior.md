# ExecuteCommandOnScrollGestureBehavior

A behavior that executes a command when a scroll gesture occurs.

## Usage

```xml
<ScrollViewer xmlns="https://github.com/avaloniaui"
              xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml">
    <Interaction.Behaviors>
        <ExecuteCommandOnScrollGestureBehavior Command="{Binding ScrollGestureCommand}" />
    </Interaction.Behaviors>
</ScrollViewer>
```
