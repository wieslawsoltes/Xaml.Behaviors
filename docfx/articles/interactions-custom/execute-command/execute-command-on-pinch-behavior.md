# ExecuteCommandOnPinchBehavior

A behavior that executes a command when a pinch gesture occurs.

## Usage

```xml
<Image xmlns="https://github.com/avaloniaui"
       xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml">
    <Interaction.Behaviors>
        <ExecuteCommandOnPinchBehavior Command="{Binding PinchCommand}" />
    </Interaction.Behaviors>
</Image>
```
