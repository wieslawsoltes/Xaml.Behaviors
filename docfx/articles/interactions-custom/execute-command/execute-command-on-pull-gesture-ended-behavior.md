# ExecuteCommandOnPullGestureEndedBehavior

A behavior that executes a command when a pull gesture ends.

## Usage

```xml
<ListBox xmlns="https://github.com/avaloniaui"
         xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml">
    <Interaction.Behaviors>
        <ExecuteCommandOnPullGestureEndedBehavior Command="{Binding PullGestureEndedCommand}" />
    </Interaction.Behaviors>
</ListBox>
```
