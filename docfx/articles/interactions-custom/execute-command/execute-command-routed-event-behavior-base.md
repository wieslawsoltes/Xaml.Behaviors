# ExecuteCommandRoutedEventBehaviorBase

Base class for behaviors that execute a command when a routed event is raised.

## Properties

| Property | Type | Description |
| --- | --- | --- |
| EventRoutingStrategy | RoutingStrategies | The routing strategy to use for the event. Default is Bubble. |
| MarkAsHandled | bool | If true, the event will be marked as handled after the command is executed. Default is true. |
