# ExecuteCommandOnScrollGestureEndedBehavior

A behavior that executes a command when a scroll gesture ends.

## Usage

```xml
<ScrollViewer xmlns="https://github.com/avaloniaui"
              xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml">
    <Interaction.Behaviors>
        <ExecuteCommandOnScrollGestureEndedBehavior Command="{Binding ScrollGestureEndedCommand}" />
    </Interaction.Behaviors>
</ScrollViewer>
```
