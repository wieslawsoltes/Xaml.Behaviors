# ExecuteCommandOnPointerWheelChangedBehavior

A behavior that executes a command when the pointer wheel changes.

## Usage

```xml
<ScrollViewer xmlns="https://github.com/avaloniaui"
              xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml">
    <Interaction.Behaviors>
        <ExecuteCommandOnPointerWheelChangedBehavior Command="{Binding WheelChangedCommand}" />
    </Interaction.Behaviors>
</ScrollViewer>
```
