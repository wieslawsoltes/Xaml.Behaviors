# ExecuteCommandOnPointerTouchPadGestureRotateBehavior

A behavior that executes a command when a touch pad rotate gesture occurs.

## Usage

```xml
<Image xmlns="https://github.com/avaloniaui"
       xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml">
    <Interaction.Behaviors>
        <ExecuteCommandOnPointerTouchPadGestureRotateBehavior Command="{Binding RotateCommand}" />
    </Interaction.Behaviors>
</Image>
```
