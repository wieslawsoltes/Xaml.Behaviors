# TextBox

The `Avalonia.Xaml.Interactions.Custom` package provides behaviors for working with `TextBox` in Avalonia.

## Behaviors

| Behavior | Description |
| --- | --- |
| [AutoSelectBehavior](auto-select-behavior.md) | Automatically selects the entire content of the associated `TextBox` when it is loaded. |
| [TextBoxSelectAllOnGotFocusBehavior](text-box-select-all-on-got-focus-behavior.md) | A behavior that allows to select all `TextBox` text on got focus event. |
| [TextBoxSelectAllTextBehavior](text-box-select-all-text-behavior.md) | Selects all `TextBox` text when attached to visual tree. |
