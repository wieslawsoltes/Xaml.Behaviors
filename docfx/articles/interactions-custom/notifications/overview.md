# Notifications

The `Avalonia.Xaml.Interactions.Custom` package provides actions and behaviors for working with notifications in Avalonia.

## Actions

| Action | Description |
| --- | --- |
| [CloseNotificationAction](close-notification-action.md) | Closes the specified `NotificationCard`. |
| [ShowErrorNotificationAction](show-error-notification-action.md) | Shows an error notification using an `INotificationManager`. |
| [ShowInformationNotificationAction](show-information-notification-action.md) | Shows an information notification using an `INotificationManager`. |
| [ShowNotificationAction](show-notification-action.md) | Shows a notification using an `INotificationManager`. |
| [ShowSuccessNotificationAction](show-success-notification-action.md) | Shows a success notification using an `INotificationManager`. |
| [ShowWarningNotificationAction](show-warning-notification-action.md) | Shows a warning notification using an `INotificationManager`. |

## Behaviors

| Behavior | Description |
| --- | --- |
| [NotificationManagerBehavior](notification-manager-behavior.md) | Provides a `INotificationManager` for the associated `Control`. |
