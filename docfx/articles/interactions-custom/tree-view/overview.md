# TreeView

The `Avalonia.Xaml.Interactions.Custom` package provides behaviors, actions and triggers for working with `TreeView` in Avalonia.

## Actions

| Action | Description |
| --- | --- |
| [ApplyTreeViewFilterAction](apply-tree-view-filter-action.md) | Filters a `TreeView` using the provided query string. |

## Behaviors

| Behavior | Description |
| --- | --- |
| [TreeViewFilterBehavior](tree-view-filter-behavior.md) | Filters `TreeView` items based on the text of a search box. |

## Triggers

| Trigger | Description |
| --- | --- |
| [TreeViewFilterTextChangedTrigger](tree-view-filter-text-changed-trigger.md) | Executes actions when the search box text changes. |

## Helpers

| Helper | Description |
| --- | --- |
| [TreeViewFilterHelper](tree-view-filter-helper.md) | Internal helper class for filtering `TreeView` items. |
