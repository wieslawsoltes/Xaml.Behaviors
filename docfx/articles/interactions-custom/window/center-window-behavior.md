# CenterWindowBehavior

Centers the attached window on the current screen when it is attached to the visual tree.

## Properties

| Property | Type | Description |
| --- | --- | --- |
| Window | `Window` | Gets or sets the window to center. If not set, the visual root window is used. |

## Usage

```xml
<Window>
    <Interaction.Behaviors>
        <CenterWindowBehavior />
    </Interaction.Behaviors>
</Window>
```
