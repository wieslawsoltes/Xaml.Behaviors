# ThemeVariant

The `Avalonia.Xaml.Interactions.Custom` package provides behaviors and triggers for working with `ThemeVariant` in Avalonia.

## Behaviors

| Behavior | Description |
| --- | --- |
| [ThemeVariantBehavior](theme-variant-behavior.md) | Sets the `ThemeVariantScope.RequestedThemeVariant` on the associated control. |

## Triggers

| Trigger | Description |
| --- | --- |
| [ThemeVariantTrigger](theme-variant-trigger.md) | Executes actions when the associated control's `StyledElement.ActualThemeVariant` matches the specified `ThemeVariant`. |
