# Show

The `Avalonia.Xaml.Interactions.Custom` package provides behaviors for showing controls in response to events.

## Behaviors

| Behavior | Description |
| --- | --- |
| [ShowBehaviorBase](show-behavior-base.md) | Base class for behaviors that show a control in response to an event. |
| [ShowOnDoubleTappedBehavior](show-on-double-tapped-behavior.md) | A behavior that allows to show a control on double tapped event. |
| [ShowOnKeyDownBehavior](show-on-key-down-behavior.md) | A behavior that allows to show a control on key down event. |
| [ShowOnTappedBehavior](show-on-tapped-behavior.md) | A behavior that allows to show a control on tapped event. |
