# ToolTips

The `Avalonia.Xaml.Interactions.Custom` package provides actions and triggers for working with `ToolTip` in Avalonia.

## Actions

| Action | Description |
| --- | --- |
| [HideToolTipAction](hide-tool-tip-action.md) | Hides the `ToolTip` of the associated or target control when executed. |
| [SetToolTipTipAction](set-tool-tip-tip-action.md) | Sets the `ToolTip.TipProperty` of the associated or target control when executed. |
| [ShowToolTipAction](show-tool-tip-action.md) | Shows the `ToolTip` of the associated or target control when executed. |

## Triggers

| Trigger | Description |
| --- | --- |
| [ToolTipClosingTrigger](tool-tip-closing-trigger.md) | Trigger that listens for the `ToolTip.ToolTipClosingEvent`. |
| [ToolTipOpeningTrigger](tool-tip-opening-trigger.md) | Trigger that listens for the `ToolTip.ToolTipOpeningEvent`. |
