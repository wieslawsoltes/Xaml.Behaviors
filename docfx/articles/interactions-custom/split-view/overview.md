# SplitView

The `Avalonia.Xaml.Interactions.Custom` package provides behaviors, actions and triggers for working with `SplitView` in Avalonia.

## Actions

| Action | Description |
| --- | --- |
| [SplitViewTogglePaneAction](split-view-toggle-pane-action.md) | Toggles the `SplitView.IsPaneOpen` state of a `SplitView` when executed. |

## Behaviors

| Behavior | Description |
| --- | --- |
| [SplitViewStateBehavior](split-view-state-behavior.md) | Updates `SplitView` properties based on size conditions. |
| [SplitViewStateSetter](split-view-state-setter.md) | Describes a state for `SplitViewStateBehavior` using size conditions. |

## Triggers

| Trigger | Description |
| --- | --- |
| [SplitViewPaneClosedTrigger](split-view-pane-closed-trigger.md) | Executes actions when the `SplitView.PaneClosed` event is raised. |
| [SplitViewPaneClosingTrigger](split-view-pane-closing-trigger.md) | Executes actions when the `SplitView.PaneClosing` event is raised. |
| [SplitViewPaneOpenedTrigger](split-view-pane-opened-trigger.md) | Executes actions when the `SplitView.PaneOpened` event is raised. |
| [SplitViewPaneOpeningTrigger](split-view-pane-opening-trigger.md) | Executes actions when the `SplitView.PaneOpening` event is raised. |
