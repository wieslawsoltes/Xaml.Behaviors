# FadeInBehavior

A simple, ready-to-use behavior that fades in the control.

### Properties
*   `Duration`: The duration of the fade-in (default: 250ms).
*   `InitialDelay`: Delay before the animation starts (default: 500ms).

### Example

```xml
<Border Background="Red" Width="100" Height="100">
    <Interaction.Behaviors>
        <FadeInBehavior Duration="00:00:01" InitialDelay="00:00:00.5"/>
    </Interaction.Behaviors>
</Border>
```
