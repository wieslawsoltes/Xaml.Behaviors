# ReleasePointerCaptureAction

Releases the pointer capture from the associated `InputElement`.

### Usage Example

```xml
<Border Background="Red">
    <Interaction.Behaviors>
        <EventTriggerBehavior EventName="PointerReleased">
            <ReleasePointerCaptureAction />
        </EventTriggerBehavior>
    </Interaction.Behaviors>
</Border>
```
