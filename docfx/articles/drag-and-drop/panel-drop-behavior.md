# PanelDropBehavior

`PanelDropBehavior` allows dropping controls dragged with `PanelDragBehavior`. It handles removing the control from its original parent and adding it to the target panel.

## Usage

```xml
<Canvas Background="LightGray" Width="300" Height="300">
    <Interaction.Behaviors>
        <PanelDropBehavior />
    </Interaction.Behaviors>
</Canvas>
```
