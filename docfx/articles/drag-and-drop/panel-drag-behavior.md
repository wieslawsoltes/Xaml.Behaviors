# PanelDragBehavior

`PanelDragBehavior` starts a drag operation for a control so it can be moved between panels. The control itself is used as the drag context.

## Usage

```xml
<Rectangle Fill="Blue" Width="50" Height="50">
    <Interaction.Behaviors>
        <PanelDragBehavior />
    </Interaction.Behaviors>
</Rectangle>
```
