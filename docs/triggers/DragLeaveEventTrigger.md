# DragLeaveEventTrigger

Namespace: `Avalonia.Xaml.Interactions.Events`

Triggers actions when a drag operation leaves the element.



## Key Properties
None

## XAML Usage
```xaml
<Interaction.Triggers>
  <Avalonia.Xaml.Interactions.Events:DragLeaveEventTrigger>
    <!-- actions -->
  </Avalonia.Xaml.Interactions.Events:DragLeaveEventTrigger>
</Interaction.Triggers>
```
