# PinchGestureTrigger

Namespace: `Avalonia.Xaml.Interactions.Custom`

Triggers actions during a pinch gesture.



## Key Properties
None

## XAML Usage
```xaml
<Interaction.Triggers>
  <Avalonia.Xaml.Interactions.Custom:PinchGestureTrigger>
    <!-- actions -->
  </Avalonia.Xaml.Interactions.Custom:PinchGestureTrigger>
</Interaction.Triggers>
```
