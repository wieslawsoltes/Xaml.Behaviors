# TappedGestureTrigger

Namespace: `Avalonia.Xaml.Interactions.Custom`

Triggers actions on a simple tap gesture.



## Key Properties
None

## XAML Usage
```xaml
<Interaction.Triggers>
  <Avalonia.Xaml.Interactions.Custom:TappedGestureTrigger>
    <!-- actions -->
  </Avalonia.Xaml.Interactions.Custom:TappedGestureTrigger>
</Interaction.Triggers>
```
