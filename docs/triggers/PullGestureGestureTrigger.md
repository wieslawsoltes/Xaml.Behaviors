# PullGestureGestureTrigger

Namespace: `Avalonia.Xaml.Interactions.Custom`

Triggers actions during a pull gesture.



## Key Properties
None

## XAML Usage
```xaml
<Interaction.Triggers>
  <Avalonia.Xaml.Interactions.Custom:PullGestureGestureTrigger>
    <!-- actions -->
  </Avalonia.Xaml.Interactions.Custom:PullGestureGestureTrigger>
</Interaction.Triggers>
```
