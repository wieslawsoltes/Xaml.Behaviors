# RightTappedGestureTrigger

Namespace: `Avalonia.Xaml.Interactions.Custom`

Triggers actions on a right-tap gesture.



## Key Properties
None

## XAML Usage
```xaml
<Interaction.Triggers>
  <Avalonia.Xaml.Interactions.Custom:RightTappedGestureTrigger>
    <!-- actions -->
  </Avalonia.Xaml.Interactions.Custom:RightTappedGestureTrigger>
</Interaction.Triggers>
```
