# PullGestureEndedGestureTrigger

Namespace: `Avalonia.Xaml.Interactions.Custom`

Triggers actions when a pull gesture ends.



## Key Properties
None

## XAML Usage
```xaml
<Interaction.Triggers>
  <Avalonia.Xaml.Interactions.Custom:PullGestureEndedGestureTrigger>
    <!-- actions -->
  </Avalonia.Xaml.Interactions.Custom:PullGestureEndedGestureTrigger>
</Interaction.Triggers>
```
