# ScreensChangedTrigger

Namespace: `Avalonia.Xaml.Interactions.Custom`

Triggers actions when the available screens change.



## Key Properties
None

## XAML Usage
```xaml
<Interaction.Triggers>
  <Avalonia.Xaml.Interactions.Custom:ScreensChangedTrigger>
    <!-- actions -->
  </Avalonia.Xaml.Interactions.Custom:ScreensChangedTrigger>
</Interaction.Triggers>
```
