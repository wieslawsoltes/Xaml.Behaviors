# ActualThemeVariantChangedBehavior

Example: [ActualThemeVariantChangedBehaviorView.axaml](samples/BehaviorsTestApplication/Views/Pages/ActualThemeVariantChangedBehaviorView.axaml)

None

<Control>

    <:ActualThemeVariantChangedBehavior/>
  </Interaction.Behaviors>
</Control>
