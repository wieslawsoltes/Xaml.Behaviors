# ShowOnTappedBehavior

Namespace: `Avalonia.Xaml.Interactions.Custom`

Shows the target control when it is tapped.



## Key Properties
None

## XAML Usage
```xaml
<Control>
  <Interaction.Behaviors>
    <Avalonia.Xaml.Interactions.Custom:ShowOnTappedBehavior/>
  </Interaction.Behaviors>
</Control>
```
