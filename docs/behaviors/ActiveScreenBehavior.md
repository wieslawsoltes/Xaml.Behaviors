# ActiveScreenBehavior

Namespace: `Avalonia.Xaml.Interactions.Custom`

Provides the currently active screen for a window.



## Key Properties
None

## XAML Usage
```xaml
<Control>
  <Interaction.Behaviors>
    <Avalonia.Xaml.Interactions.Custom:ActiveScreenBehavior/>
  </Interaction.Behaviors>
</Control>
```
