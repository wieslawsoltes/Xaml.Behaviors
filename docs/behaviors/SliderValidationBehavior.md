# SliderValidationBehavior

Namespace: `Avalonia.Xaml.Interactions.Custom`

Example: [SliderValidationBehaviorView.axaml](samples/BehaviorsTestApplication/Views/Pages/SliderValidationBehaviorView.axaml)

<Control>
