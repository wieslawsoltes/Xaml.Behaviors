# ItemDragBehavior

Namespace: `Avalonia.Xaml.Interactions.Draggable`

Enables reordering of items in an ItemsControl by dragging and dropping items.



## Key Properties
None

## XAML Usage
```xaml
<Control>
  <Interaction.Behaviors>
    <Avalonia.Xaml.Interactions.Draggable:ItemDragBehavior/>
  </Interaction.Behaviors>
</Control>
```
