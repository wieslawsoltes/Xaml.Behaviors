# ListBoxUnselectAllBehavior

Namespace: `Avalonia.Xaml.Interactions.Custom`

Clears the selection in a ListBox.



## Key Properties
None

## XAML Usage
```xaml
<Control>
  <Interaction.Behaviors>
    <Avalonia.Xaml.Interactions.Custom:ListBoxUnselectAllBehavior/>
  </Interaction.Behaviors>
</Control>
```
